## 🎯 布局特点

### 左侧区域 (55%)
- **上方**: 歌词显示区域 (占左侧大部分空间)
- **下方**: 播放控制按钮 + 进度条

### 右侧区域 (45%)  
- **Header**: 返回按钮、歌曲信息、设置按钮
- **专辑图片**: 居中显示，带动画效果
- **更多按钮**: 收藏、分享等功能

## 🚗 车机使用优势

### 1. 歌词阅读优化
- 歌词占据最大显示区域
- 左侧位置符合阅读习惯
- 滚动歌词清晰可见

### 2. 操作便捷性
- 播放控制在底部，易于触达
- 进度条与播放按钮在同一区域
- 专辑图片独立显示，不干扰操作

### 3. 视觉层次
- 歌词 (主要信息) - 左上
- 控制 (操作区域) - 左下  
- 图片 (视觉元素) - 右侧

## 🔧 技术实现

### 布局结构
```typescript
<View style={styles.container}>
  <View style={styles.left}>           // 左侧 55%
    <View style={styles.leftTop}>      // 歌词区域
      <Lyric />
    </View>
    <View style={styles.leftBottom}>   // 控制区域
      <Player />
    </View>
  </View>
  <View style={styles.right}>          // 右侧 45%
    <Header />
    <View style={styles.rightContent}>
      <MoreBtn />
      <Pic />
    </View>
  </View>
</View>
```

### 样式配置
- `left`: 55%宽度，纵向布局
- `leftTop`: flex: 1，占据剩余空间
- `leftBottom`: 固定高度，底部对齐
- `right`: 45%宽度，专辑图片居中
